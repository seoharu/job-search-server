# Job Search Server

Backend server for job search platform using Express.js and MySQL.

## Tech Stack

- Node.js
- Express.js
- MySQL
- Sequelize (ORM)
- JWT Authentication
- Swagger Documentation

## Requirements

- Node.js 18.x or higher
- MySQL 8.x
- npm or yarn
